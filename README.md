# erode

An experimental NodeJS-like JavaScript & TyprScript runtime environment built on V8 and Rust.

## Implemented APIs:

- console.log
- console.error
- readFile
- writeFile
- removeFile
- fetch (very basic)
- setTimeout

### To run the example file

```sh
cargo run ./src/example.ts
```
